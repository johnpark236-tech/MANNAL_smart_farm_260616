# 영농 도우미 Android 앱

PC 서버 없이 스마트폰에서 직접 공개 화면을 조회하고, 영농일지를 스마트폰에 저장하는 Android 앱입니다.

## 기능

- 시세: API 인증키 없이 경기도농업기술원 공개 가격정보 조회
- 날씨: 기상청 공개 화면 기준 10일 상세 전망 + 4주 경향 전망 조회
- 영농일지: 날짜, 작물, 작업 내용, 메모를 스마트폰 내부 저장소에 저장
- TXT 백업: 영농일지 전체 기록을 텍스트 파일로 내보내기
- 글씨 크기 조절: 화면 상단 `-`, `+` 버튼으로 전체 글씨 크기 조절
- 큰 글씨 줄바꿈: 안내문, 결과 카드, 표 셀이 자동 줄바꿈되도록 구성
- 파스텔 농장풍 UI: 밝은 그라데이션 배경, 둥근 버튼, 부드러운 카드 스타일 적용
- 화면 이름: `만날 농사-시세`, `만날 농사-날씨`, `만날 농사-영농일지`
- 앱 아이콘: 초록색 무한대 형태 아이콘 적용
- 상태바 안전 여백: 스마트폰 시계/와이파이 영역과 앱 내용이 겹치지 않도록 상단 여백 적용

## 빌드

Android SDK가 설치된 Windows PC에서 실행합니다.

```powershell
.\build_apk.ps1
```

생성 파일:

```text
out\AgriPrice.apk
```

## 영농일지 백업

영농일지 화면에서 `TXT 백업` 버튼을 누르면 전체 일지를 다음 형식의 파일로 저장합니다.

```text
farm_journal_yyyyMMdd_HHmm.txt
```

Android 10 이상은 다운로드 폴더에 저장됩니다.
Android 9 이하에서는 앱 전용 문서 폴더에 저장됩니다.

## 설치

생성된 APK를 스마트폰으로 옮긴 뒤 설치합니다.
직접 만든 APK이므로 Android에서 "알 수 없는 앱 설치" 허용이 필요할 수 있습니다.
